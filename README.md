## 🏗️ What I've Built

A set of lightweight web tools designed around real workflows in the construction supply chain:

| Tool | What it does |
|------|-------------|
| [**Vendor Discovery**](https://qwik-vendor-discovery.netlify.app/) | Fast vendor search and discovery for construction procurement |
| [**Catalogue Hub**](https://mob-catalogue-hub.netlify.app/) | Centralized product catalogue browser for the MOB ecosystem |
| [**Quote Builder**](https://mob-quotebuilder.netlify.app/) | Streamlined quote generation for field and sales teams |
| [**Price Comparison Dashboard**](https://price-comparison-dashboard.netlify.app/) | Cross-vendor price intelligence at a glance |
| [**Qwik Products Dashboard**](https://mob-qwikproducts-dashboard.netlify.app/) | Real-time product and inventory visibility |

These started as internal experiments. They exist because the problem was real and the existing options were overkill.
